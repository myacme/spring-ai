# 快速启动指南

## 第一步：配置 API Key

### 方法一：修改配置文件
编辑 `src/main/resources/application.yml`，将 API Key 替换为你的 DeepSeek API Key：

```yaml
spring:
  ai:
    dashscope:
      api-key: sk-your-actual-api-key-here
```

### 方法二：设置环境变量（推荐）
```bash
# Windows PowerShell
$env:DEEPSEEK_API_KEY="sk-your-actual-api-key-here"

# Windows CMD
set DEEPSEEK_API_KEY=sk-your-actual-api-key-here

# Linux/Mac
export DEEPSEEK_API_KEY=sk-your-actual-api-key-here
```

## 第二步：启动应用

### 方式一：使用启动脚本（推荐）
双击运行 `start.bat`

### 方式二：命令行启动
```bash
cd E:\project\Java\aa-myproject\spring-ai\agent\alibaba
mvn clean compile spring-boot:run
```

等待看到以下输出表示启动成功：
```
Started AlibabaAiApplication in X.XXX seconds
```

## 第三步：测试 API

### 方式一：使用测试脚本（推荐）
双击运行 `test-api.bat`

### 方式二：手动测试

#### 1. 健康检查
打开浏览器访问：http://localhost:8083/api/ai/health

应该看到：`Spring AI Alibaba + DeepSeek is running!`

#### 2. 使用 Postman 或 curl 测试

**普通聊天：**
```bash
curl -X POST http://localhost:8083/api/ai/chat ^
  -H "Content-Type: application/json" ^
  -d "{\"message\": \"你好\"}"
```

**文本分词：**
```bash
curl -X POST http://localhost:8083/api/splitter/token ^
  -H "Content-Type: application/json" ^
  -d "{\"text\": \"这是一段测试文本。用于演示分词功能。Spring AI Alibaba 提供了强大的文本处理能力。\"}"
```

**天气查询（Function Calling）：**
```bash
curl -X POST http://localhost:8083/api/weather/query ^
  -H "Content-Type: application/json" ^
  -d "{\"message\": \"北京今天天气怎么样？\"}"
```

## 常见问题排查

### 问题1：依赖下载失败
**解决方案：** 确保网络连接正常，Maven 可以访问 Spring Milestones 仓库。

### 问题2：找不到 ChatClient 类
**解决方案：** 
1. 检查 pom.xml 中是否包含 spring-ai-alibaba-starter 依赖
2. 确认已配置 Spring Milestones 和 Snapshots 仓库
3. 执行 `mvn clean install` 重新下载依赖

### 问题3：API Key 无效
**解决方案：** 
1. 确认 API Key 格式正确（以 sk- 开头）
2. 检查环境变量是否正确设置
3. 验证 API Key 是否过期

### 问题4：端口被占用
**解决方案：** 修改 `application.yml` 中的端口号
```yaml
server:
  port: 8084  # 改为其他可用端口
```

## 核心功能速览

| 功能 | 接口路径 | 说明 |
|------|---------|------|
| 健康检查 | GET /api/ai/health | 检查服务状态 |
| 普通聊天 | POST /api/ai/chat | 基础对话功能 |
| 流式聊天 | POST /api/ai/chat/stream | 流式响应 |
| Token分词 | POST /api/splitter/token | 基于Token的分词 |
| 句子分词 | POST /api/splitter/sentence | 基于句子的分词（推荐中文） |
| 天气查询 | POST /api/weather/query | Function Calling示例 |
| 系统提示词 | POST /api/advanced/chat-with-system | 自定义系统提示 |
| 角色扮演 | POST /api/advanced/role-play | 多角色切换 |
