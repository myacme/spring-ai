# 天气查询Agent测试用例

## 测试准备
1. 确保已配置DeepSeek API密钥
2. 启动应用：`mvn spring-boot:run`
3. 应用启动后访问：http://localhost:8082

## API测试用例

### 1. 健康检查
```bash
curl -X GET http://localhost:8082/weather/health
```

### 2. 获取API信息
```bash
curl -X GET http://localhost:8082/weather/api-info
```

### 3. 获取支持的城市列表
```bash
curl -X GET http://localhost:8082/weather/cities
```

### 4. 自然语言天气查询测试

#### 当前天气查询
```bash
curl -X POST http://localhost:8082/weather/query \
  -H "Content-Type: application/json" \
  -d '{"query": "今天北京天气怎么样？"}'
```

```bash
curl -X POST http://localhost:8082/weather/query \
  -H "Content-Type: application/json" \
  -d '{"query": "上海现在的温度是多少？"}'
```

#### 天气预报查询
```bash
curl -X POST http://localhost:8082/weather/query \
  -H "Content-Type: application/json" \
  -d '{"query": "明天广州天气预报"}'
```

```bash

  -H "Content-Type: application/json" \
  -d '{"query": "未来三天深圳天气"}'
```

#### 降水查询

#### 多城市测试
```bash

  -H "Content-Type: application/json" \
  -d '{"query": "南京今天天气如何？"}'
```


### 5. 结构化查询测试

## 自动化测试脚本

可以创建测试脚本批量执行测试用例：

```bash
#!/bin/bash
# test_weather_agent.sh

echo "开始测试天气查询Agent..."

# 健康检查
echo "1. 健康检查测试..."
curl -s http://localhost:8082/weather/health | jq '.'

# 基础查询测试
echo "2. 基础天气查询测试..."
curl -s -X POST http://localhost:8082/weather/query \
  -H "Content-Type: application/json" \
  -d '{"query": "今天北京天气"}' | jq '.'

echo "测试完成！"
```

运行测试：`chmod +x test_weather_agent.sh && ./test_weather_agent.sh`
