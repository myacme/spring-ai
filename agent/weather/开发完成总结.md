## API接口

### 核心接口
- `POST /weather/query` - 自然语言天气查询
- `POST /weather/structured` - 结构化天气查询
- `GET /weather/cities` - 获取支持的城市列表
- `GET /weather/health` - 健康检查
- `GET /weather/api-info` - API信息

### 查询示例
```bash
# 自然语言查询
curl -X POST http://localhost:8082/weather/query \
  -H "Content-Type: application/json" \
  -d '{"query": "今天北京天气怎么样？"}'

# 结构化查询
curl -X POST http://localhost:8082/weather/structured \
  -H "Content-Type: application/json" \
  -d '{
    "query": "天气查询",
    "city": "北京",
    "date": "今天",
    "queryType": "CURRENT"
  }'
```
